# Yonny's Portfolio

Personal portfolio and resume website built with React + Vite.

## About

A single-page application showcasing my profile as a Full Stack Web Developer, featuring smooth scroll navigation, a projects/experiences section, and a contact section with social links.

## Tech Stack

- **React 18** — UI components
- **Vite** — build tool with HMR
- **react-router-dom** — routing
- **react-scroll** — smooth scroll navigation

## Sections

- **About** — intro, skills summary, and resume download link
- **Experiences** — project cards (Chilling Time, Countdown, Grades, Rock Paper Scissors, Gallery, Bank Card Form, Crud University)
- **Contact** — links to Facebook, WhatsApp, Instagram, LinkedIn, GitHub, and email

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
